metafill

Fill your App Store metadata in every App Store language, fast.

A local tool for indie and solo Apple developers. Write your English listing once, machine-translate it into every locale, review it, and upload it to your App Store Connect draft. Free and open source.

Install

git clone https://github.com/elpofigisto/metafill.git
cd metafill && npm install && npm run dev

Bring your own keys. metafill runs entirely on your machine and talks directly to Apple and your AI provider. No backend, no telemetry, nothing is sent to any poTeam server.
